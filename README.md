# 书籍销售平台（BookShop System）

## 项目简介
本项目是一个基于Django开发的书籍销售平台，用于实现在线书籍浏览、搜索、购物车、订单管理以及评论等功能。

## 技术栈
- 后端：Python / Django
- 前端：HTML / CSS / JavaScript
- 数据库：SQLite
- 工具：Git / GitHub

## 功能模块
- 用户注册与登录
- 书籍分类与搜索
- 购物车功能
- 订单管理
- 评论系统

## 个人职责
- 负责系统需求分析
- 数据库设计（用户表、书籍表、订单表等）
- 后端接口开发（Django ORM实现）
- 前端页面开发与交互

## 项目收获
通过该项目，熟悉了Django开发流程、前后端交互方式以及数据库设计，提高了独立开发能力。
